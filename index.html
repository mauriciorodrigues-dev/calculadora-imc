Índice de Massa Corporal

O IMC é uma medida que relaciona o peso e a altura de uma pessoa para estimar seu grau de obesidade ou magreza e, consequentemente, avaliar o risco para a saúde. É uma ferramenta simples e amplamente utilizada para categorizar indivíduos em faixas de peso saudável, sobrepeso ou obesidade. O cálculo do IMC é feito pela seguinte fórmula:

IMC = Peso (em quilogramas) / (Altura (em metros) * Altura (em metros))
